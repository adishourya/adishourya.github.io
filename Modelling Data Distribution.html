Just remeber the 68 - 95 -99.7 rule

68 % of the total sample or pop falls b/n → ( μ - σ <= μ <= μ + σ )
95 % of the total sample or pop falls b/n → ( μ - 2σ <= μ <= μ + 2σ )
99.7 % of the total sample or pop falls b/n → ( μ - 3σ <= μ <= μ + 3σ )
