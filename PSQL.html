What is a database ?

  • A database in SQL server is made up of a collection of tables that stores a set of structured data
  • A table contains collection of rows called records and columns which is also reffered as attributes

Create Databse


CREATE DATABASE test; -- creates a databse named test

connect to a database

psql --host localhost -U postgres -d test

or just \c test from the psql shell

Delete Database

DROP DATABASE test;

How to create a basic table
