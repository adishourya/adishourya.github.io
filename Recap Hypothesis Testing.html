For Proportion

step 1 : Make Hypothesis and choose significance level step 2 : calculate p_sample step 3 : calculate σ of sample dist of sample prop = σ_p_hat

σ_p_hat = √(p₀*q₀/n) :: where p₀ and q₀ are from H₀

step4 : calculate mod(z) | z | = | p₀ - p_sample | / σ_p_hat

step 5: calculate p_val . check if 1-tailed or 2-tailed 

import scipy.stats as st
# if 1-tailed
p_val = st.norm.sf(abs(z))

# if 2-tailed
p_val = st.norm.sf(abs(z)) * 2

step 6 : if p_val < α : reject H₀

For Means

step1 : Make hypothesis and choose significance level

step2 : calculate test statistics x_bar ans sx
x_bar → mean of sample , sx → σ of sample

step 3 :to calcultate z-stat = x_bar - μ₀ / std dev of sample dist of sample pop

i.e z = x_bar - μ₀ / σx mostly σx cannot be calculatead as σx = σ_population / √n and σ_pop is rarely known

∴ we approx σ_population with sx we use t-stat instead of z-stat

when :
  • we dont know the true std dev of pop.
  • low sample size
so this test statistics becomes a t-test
t = x_bar - μ₀ / σx t = x_bar - μ₀ / (sx/√n)

step 6 : calculate p_val 

import scipy.stats as st
# if 1-tailed
p_val = st.t.sf(abs(t),n-1)

# if 2-tailed
p_val = st.t.sf(abs(t),n-1) * 2

step 6 : if p_val < α : reject H₀
