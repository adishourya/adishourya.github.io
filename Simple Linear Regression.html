r = (1/n-1) * (zx*zy)
where zx and zy are z-scores

equation of a Trend line

y_hat = β0 * x + β1

where
β0 = r * (σy / σx)

now if we know β0 we can calculate β1

  • as we know for a least squared reg line must pass through (μx , μy)

μy = r*(σy/σx) + β1

R² (the big boy)

  • in a way,r, squared measures how much prediction error is eliminated when we use least-squares regression.
  • comapred to the worst trend line which would be r(correlation coefficient) = 0.
  • y_hat = r*(sy/sx) *x + c | @r=0 ⇒ y_hat = c
  • now a trend line always passes through y_avg and x_avg .substituting
  • gives us y_hat = y_avg
how much % squared err was removed compared to y_avg
r² = err₁ - err₂ / err₁

where err₁ = squared err of trend line where err₂ = squared err of y_avg line

Many formal definitions say that r, squared tells us what percent of the variability in the y variable is accounted by the regression line

  • When our model we have multiple independent variables, we use Adjusted R-Squared instead of R-Squared value for model evaluation.
  • Because when we have more independent variable in R-Squared calculation(weather they are correlated or not correlated), R-squared value automatically get increased due to slope coefficient value. R Squared Formula
  • In Adjusted R-Squared calculations, it penalizes the independent variables which are not correlated to the dependent variable. So there is no impact of independent variable which are not correlated to dependent variable. So in our Multiple Regression cases we use Adjusted R square for model accuracy check instead of R-Square.
