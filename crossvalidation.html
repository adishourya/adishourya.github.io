HELPS TO DETERMINE WHICH ML LEARNING STRATERGY WOULD BE BEST

  1. Cross validation allows us to compare different machine learning methods and get a sense of how well they will work in practice
  1. Usually in ML the dataset is divided into a training set and a testing set But this set is chosen randomly .
  1. A key challenge with machine learning in general, is that we canâ€™t know how well our model will perform on new data until we actually test it.

TO ADDRESS THIS, WE CAN SPLIT OUR INITIAL DATASET INTO SEPARATE TRAINING AND TEST SUBSETS.

  • To partition the data into a number of subsets say n
  • Hold out a set at a time and train the model on remaining set i.e n-1
  • Test model on hold out set
  • Repeat for all such sets (n times).
  • And in the End it gives all the (n) scores.
  • Average the result and choose the preferred model.
  1. Stratified Technique :: In this technique each such subset would have the same proportion of the data . For e.g if the dataset has 80% male and the rest female. Then all such subset would have 80% male and 20% female.
